Installation Guide

  1. Download the APK – Tap the big red button above.
  2. Open the file – Once downloaded, open it from your notifications or Downloads folder.
  3. Tap “Install” – If Android asks for permission, allow installation from unknown sources (see next step).
  4. Enable “Install unknown apps” – If blocked, go to Settings → Security → toggle on “Allow from this source” for your browser or file manager.
  5. Open the app – After installation, tap “Open” and start learning.
